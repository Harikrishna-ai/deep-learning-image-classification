# Deep Learning Image Classification

## Overview

This project is based on my deep learning coursework where I applied deep learning techniques to an image classification problem. The main aim was to compare a custom CNN model with a transfer learning approach.

---

## Objective

* Build a CNN model from scratch
* Use a pre-trained model (Transfer Learning)
* Compare performance of both models

---

## Dataset

The dataset consists of images (e.g., cats and dogs).
The images were resized and normalised before training.

---

## Models Used

### CNN Model

* Built using convolutional layers
* Used as a baseline model

### Transfer Learning Model

* Used VGG16 pre-trained model
* Gave better accuracy compared to CNN

---
